.. _setup::

Setup PyChannel
===============

Installing PyChannel is incredibly easy. Just download the package and untar it or clone it from github. Then
cd into the directory that you put PyChannel into and run ``python setup.py install`` to install the module.

Once the install is done running run ``./pychannel-serve`` from the root of the PyChannel directory. Thats it
PyChannel should be running on port 80!

.. note::
   
   You can supply a port to the ``./pychannel-serve`` command to serve on a different port.
