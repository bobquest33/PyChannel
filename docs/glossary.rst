.. _glossary:


Glossary
========

.. glossary::

   Board
      A collection of threads that are usually organized over some theme.

   Thread
      A post which belongs to a board and can be replied to.

   Reply
      A post which belongs to a thread. The lowest level in the post hirearchy.

   Bump
      When a reply to a thread is made, it is pushed to the top of a board or 'bumped'.

   Pruning
      Boards have a limit on the number threads that they can contain. When the number of
      threads exceeds the maximum number the least recently bumped threads are deleted or 'pruned'.
